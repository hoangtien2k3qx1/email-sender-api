# Email-sender-api

## Introduction

Email-Sender API trong ngữ cảnh của một ứng dụng Java Backend là một phần mềm cung cấp một giao diện lập trình ứng dụng (API) để gửi email từ phía máy chủ hoặc server của ứng dụng.
